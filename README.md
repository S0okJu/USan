# USan

GPS 기반 중고거래 플랫폼입니다. 오프라인 거래 시 구매자와 판매자가 일정 범위 내에 있으면 결제를 수행할 수 있습니다.

## 기술 스택

- JAVA
- Android Studio
- Python == 3.9.16
- Flask == 2.2.3
- Firebase
